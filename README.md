### Structure.

```sh
.
│   .editorconfig
│   .eslintrc.js
│   .gitignore
│   favicon.ico
│   package.json
│   README.md
│   webpack.config.js
│   yarn.lock
│
├───dist
│   │   favicon.ico
│   │   index.html
│   │
│   └───js
│           canvas.bundle.js
│           canvas.bundle.js.map
│
└───src
    │   index.html
    │
    └───js
            bang-circle.js
            canvas.js
            circle.js
            design.js
            fixed-circle.js
            html-canvas.js
            utils.js
```

### Available Scripts.
Use the package manager <i>[yarn](https://yarnpkg.com/)</i> or  <i>[npm](https://www.npmjs.com/)</i>
```sh
> yarn run dev
> yarn run start
> yarn run test
> yarn run build
> yarn run deploy
```
